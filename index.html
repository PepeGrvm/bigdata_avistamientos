Inicio en Google Cloud Platform

Comenzamos nuestro proyecto en Google Cloud Platform, donde gestionaremos todos los recursos.

Gestión de permisos

Configuramos los permisos necesarios para elaborar todos los procesos.

Creación del Bucket

Comenzamos creando un bucket en Google Cloud Storage, el cual servirá como contenedor para nuestros archivos de datos.

Bucket creado

El bucket fue creado exitosamente y quedó disponible para almacenar archivos CSV.

Subida del archivo CSV

Subimos los datos crudos en formato CSV a nuestro bucket para poder procesarlos más adelante.

Creación del dataset

Definimos un nuevo dataset en BigQuery donde importaremos y analizaremos los datos.

Inicio en Dataprep

Iniciamos el proceso de preparación de datos utilizando Dataprep, una herramienta visual para limpiar y transformar datos.

Conexión de Dataprep con Cloud Storage

Dataprep se conecta al bucket para acceder al CSV subido y preparar los datos de manera visual.

Vista previa de datos

Exploramos los datos en Dataprep para entender su estructura y posibles problemas de calidad.

Tratamiento de datos

Aplicamos transformaciones, limpieza y formateo para dejar los datos listos para el análisis.

Flujo de procesamiento

Construimos un flujo de trabajo dentro de Dataprep para automatizar los pasos de tratamiento.

Publicación a BigQuery

Publicamos el resultado del tratamiento directamente en BigQuery para su análisis.

Inicio en BigQuery

Comenzamos el análisis de los datos tratados utilizando BigQuery, la base de datos analítica de Google Cloud.

Primera Consulta

Realizamos la primera consulta para comprobar la correcta carga y estructura de los datos tratados.

Segunda Consulta

Exploramos patrones y agrupaciones clave para alimentar la visualización.

Tercera Consulta

Generamos métricas y tablas finales que luego serán utilizadas en Looker Studio.

Inicio en Looker Studio

Iniciamos el proceso de creación de visualizaciones utilizando Looker Studio, la herramienta de BI de Google.

Integración con Looker Studio

Agregamos los datos desde BigQuery a Looker Studio para la creación del dashboard final.

Primer gráfico

Comenzamos a crear visualizaciones para mostrar los avistamientos de las 5 formas más vistas

Segundo gráfico

Comparamos métricas relevantes en una visualización, viendo las cantidades de avistamientos por año

Tercer gráfico

Incorporamos una manera más interactiva para poder ver la cantidad de avistamientos por estado dentro de los Estados Unidos

Dashboard final

Este dashboard muestra visualizaciones clave sobre la cantidad de avistamientos de ovnis en los Estados Unidos, incluyendo las formas más comunes, su distribución por estado y la evolución anual.
